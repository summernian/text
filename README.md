·git commit 提交
  -- ps吐槽: git bash的编辑器对新人太不友好了
  -- 提交后，按下小写字母i，会进入编辑模式。
  -- 可以在此模式下输入你想要的commit message。
  -- 输入结束以后，按下esc退出编辑模式。
  -- 这时按下英文输入法下的冒号，再输入wq，就可以保存退出了。
  -- w是write，q是quit。
  -- 也可以在按esc退出编辑模式以后，切换到大写模式，连按两下Z。
